Aplicativo Modelo

Aplicativo que servirá de modelo para o desenvolvimento de novos aplicativo web utilizando o framework Flask!



Passos para a inicialização:

  1. Instalar o Pipenv: pip install pipenv
  2. Criar uma nova pasta: md nova_pasta
  3. Entra na nova pasta: cd nova_pasta
  4. Criar uma ambiente virtual para o projeto: pipenv install novo_projeto
  5. Iniciar o ambiente virtual: pipenv shell
  6. Instalar os pacotes do projeto: pipenv install flask flask-wtf flask-sqlalchemy flask-bcrypt flask-login Pillow flask-mail
  7. Copia o 'projetomodelo' para a nova_pasta
  8. Cria o banco de dados no Mysql
  9. Ajustar as configuração do banco de dados no arquivo config.py
  10. Executar o aplicativo: python run.py
